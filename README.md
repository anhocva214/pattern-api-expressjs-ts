# TEMPLATE NODEJS 
Template NodeJs (Express Js) With Typescript

### Feature:
    ✅ Controller
    ✅ Middleware
    ✅ Model
    ✅ Repository

### Install   
```bash
yarn install
```

### Run development
```bash
yarn dev
```

### Run production
```bash
yarn build & yarn start
```

### Push git tự động
Chạy lệnh sau đây:
```bash
python push.py
```
